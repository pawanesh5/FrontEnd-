What is Modernizr?

its a collection of superfast tests- or "defects" as we like to call them- which runs as your web page load, then you can use the results to tailor the experience to the user."


Why do I need it?

All web developers come up againt differences between browsers and devices. That's largely due to different features sets: the latest versions of the popular browsers can do some awsome things which older browsers can't- but we still have to support the older ones.
